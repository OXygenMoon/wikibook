# Apply Glassmorphism Style to UI Components

I will apply the user-provided "Glassmorphism" style (frosted glass effect) to other suitable floating UI components in the application to maintain a consistent and modern design language.

## 1. Target Components for Styling
Based on the analysis, I will apply this style to:

1.  **Global Toast Notifications** (`templates/base.html`)
    -   Currently uses a similar but slightly different style. I will update it to match the *exact* classes provided by the user for consistency.
    -   Key classes: `bg-stone-900/60 backdrop-blur-md text-white border-white/10 ring-1 ring-white/5`.
    -   *Note*: Since toasts are usually light/dark theme adaptive, I might need to adjust the hardcoded `bg-stone-900` to be theme-aware or stick to the user's specific dark glass look if they want a unified "dark glass" aesthetic for notifications. *Decision*: I will use the user's exact provided style for notifications as high-contrast alerts often look good in dark mode regardless of theme.

2.  **Badge Tooltip** (`templates/user/profile.html`)
    -   The tooltip showing the equipped badge name.
    -   Apply the frosted glass effect to the tooltip container.

3.  **Scroll-to-Top Button** (if exists, or I can check `base.html`)
    -   Often a floating element. If present, I will style it.

4.  **Floating Action Buttons (FAB)**
    -   Specifically the **Sticker FAB** in `templates/index.html` (the button to open the sticker drawer).
    -   It is a prime candidate for this floating glass style.

## 2. Implementation Details

### A. Global Toast (`templates/base.html`)
-   Update `showGlobalToast` JS function and the Jinja2 flash message loop.
-   Replace existing classes with: `bg-stone-900/60 backdrop-blur-md text-white border border-white/10 ring-1 ring-white/5 shadow-2xl`.

### B. Sticker FAB (`templates/index.html`)
-   Locate `#sticker-fab`.
-   Apply the same glassmorphism classes, adjusting for shape (keeping it `rounded-full`).

### C. Sticker Drawer Header (`templates/index.html`)
-   The header of the sticker drawer can also benefit from this style to look like a floating glass panel.

## 3. Execution Steps
1.  **Modify `templates/base.html`**: Update Toast notification styles (JS & Jinja2).
2.  **Modify `templates/index.html`**: Update Sticker FAB and potentially the drawer header.

This plan ensures the beautiful style you discovered is not just a one-off, but part of the app's visual identity.
