## 目标

重构用户个人主页 UI，采用现代化的 **Glassmorphism (毛玻璃)** 风格，提升视觉层次感和交互体验，打造高级的数据仪表盘。

## 变更范围

* **前端模板**: 重写 [templates/user/profile.html](file:///root/projects/wikibook/templates/user/profile.html)

## 设计方案

### 1. 视觉基调 (Visual Identity)

* **背景**: 使用柔和的渐变光晕 (Gradient Blobs) 作为底层背景，营造氛围感。

* **材质**: 核心容器采用 **Glassmorphism** 风格：

  * `bg-white/60`: 半透明白色背景

  * `backdrop-blur-xl`: 强力高斯模糊

  * `border-white/50`: 细腻的半透明边框

  * `shadow-lg`: 柔和投影

* **排版**: 增加留白 (Whitespace)，使用更精致的字体排版和微交互动画。

### 2. 布局结构

* **Profile Header (头部)**:

  * 悬浮式玻璃卡片，展示大头像（带光环）、用户名、身份标签、加入时间。

  * 右侧/下方集成“核心指标”（Wiki数、笔记数、徽章数、打卡），使用大数字 + 图标设计。

* **Detailed Stats (统计详情)**:

  * 3列网格布局，分别展示“学习习惯”、“作息规律”、“社区互动”。

  * 每个数据点配有专属图标和进度条/可视化元素。

* **Content Area (内容区)**:

  * **Tabs**: 现代胶囊式切换按钮。

  * **Achievements (成就)**:

    * 已获得徽章：彩色高亮，悬停浮起效果。

    * 未获得徽章：半透明灰阶，显示锁头图标。

  * **Activity (动态)**: 时间轴风格展示近期笔记。

### 3. 技术实现

* 仅修改 HTML 模板，利用 Tailwind CSS 类实现所有效果（无需额外 CSS 文件）。

* 保持现有的 JS 逻辑（标签切换、徽章佩戴）不变。
