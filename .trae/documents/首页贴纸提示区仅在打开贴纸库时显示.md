## 目标
- 在首页搜索栏下方、页面中间位置增加一个贴纸提示区域（虚线框），默认不占布局高度。
- 仅在用户“打开贴纸库”后显示该提示区域；关闭贴纸库后自动隐藏。
- 不改变“精选 Wiki”卡片区域高度与布局。

## 变更范围
- 模板文件：[index.html](file:///root/projects/wikibook/templates/index.html)
- 前端脚本：[sticker.js](file:///root/projects/wikibook/static/js/sticker.js)

## 实现方案
- 在搜索栏容器结束处（约第 31-33 行之间）插入一个隐藏的提示区容器：
  - 元素结构：`<div id="sticker-drop-zone" class="hidden ...">` 包含虚线边框、提示图标与文案。
  - 布局：`max-w-4xl mx-auto mt-8 mb-4 px-4`，位置居中，处于搜索栏与内容卡片之间。
  - 隐藏方式：使用 `hidden`（display: none）确保默认完全不占高度。
- 在贴纸库打开/关闭逻辑处控制该区域显隐：
  - 打开贴纸库（`openDrawer()`）时执行：`document.getElementById('sticker-drop-zone')?.classList.remove('hidden')`
  - 关闭贴纸库（`closeDrawer()`）时执行：`document.getElementById('sticker-drop-zone')?.classList.add('hidden')`
  - 位置：在 [sticker.js](file:///root/projects/wikibook/static/js/sticker.js) 的 `StickerManager` 类中，已有 `openDrawer/closeDrawer` 方法与 `isDrawerOpen/isEditing` 状态；在方法末端插入上述显隐代码。

## 代码示例
- 模板插入（位于搜索栏容器之后、列表卡片之前）：
```
<div id="sticker-drop-zone" class="hidden max-w-4xl mx-auto px-4 mt-8 mb-4">
  <div class="border-2 border-dashed border-stone-300 rounded-2xl p-6 flex flex-col items-center justify-center text-center bg-stone-50/50 min-h-[140px] hover:bg-stone-100 transition-colors">
    <div class="p-3 rounded-full bg-white shadow-sm mb-3 text-primary">
      <svg xmlns="http://www.w3.org/2000/svg" class="h-6 w-6" fill="none" viewBox="0 0 24 24" stroke="currentColor"><path stroke-linecap="round" stroke-linejoin="round" stroke-width="2" d="M12 4v16m8-8H4" /></svg>
    </div>
    <h3 class="text-base font-bold text-stone-600">拖拽贴纸到这里</h3>
    <p class="text-xs text-stone-400 mt-1">贴纸将固定在首页背景上</p>
  </div>
</div>
```
- JS 显隐控制（示意）：
```
openDrawer() {
  // 原有逻辑 ...
  const dz = document.getElementById('sticker-drop-zone');
  if (dz) dz.classList.remove('hidden');
}
closeDrawer() {
  // 原有逻辑 ...
  const dz = document.getElementById('sticker-drop-zone');
  if (dz) dz.classList.add('hidden');
}
```

## 验证与影响
- 默认首页：提示区不显示且不占位；精选 Wiki 区域高度不变。
- 打开贴纸库：提示区在搜索栏下方居中显示；关闭后恢复隐藏。
- 与现有样式栈保持一致（Tailwind 类名）；不引入第三方依赖。

## 备选方案（若不改 JS）
- 使用状态类触发：如果打开贴纸库时页面或画布会加上 `editing` 类，可在模板中通过 `hidden`/`block` 的切换类基于该状态控制显示；但为保证行为可靠，优先采用显隐的 JS 控制。
